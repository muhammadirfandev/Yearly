
Timely- Year Progress Tracker
A sleek, minimalist web application designed to visually track the passage of time throughout the year. Built with a high-performance, dark-mode aesthetic, the application provides real-time tracking down to the millisecond, helping users stay mindful of their annual goals and timelines.

Key Features:
Live Percentage Calculation: Displays the exact percentage of the current year completed with precision metrics.

Dynamic Progress Visualization: Features a smooth, linear neon progress bar reflecting the year's completion status.

Multi-Timezone Toggle: Built-in localization support with integrated quick-toggles for global time zones (Local, London, New York).

Countdown Metrics: Includes an ongoing countdown showing the exact number of days remaining until the next year.
